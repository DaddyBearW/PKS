# Практика 4

Простой проект на `ASP.NET Core MVC` и `Entity Framework Core` под `PostgreSQL`.

Что есть:

- список городов с поиском;
- страница города;
- страница достопримечательности;
- хранение данных в PostgreSQL через EF Core.

Что сделать перед запуском:

1. В `pgAdmin` создать базу данных `pks_pr4`.
2. Если нужно, поменять логин и пароль в [appsettings.json](./appsettings.json).
3. В папке проекта выполнить `dotnet restore`.
4. Потом выполнить `dotnet run`.

Таблицы и тестовые данные создаются автоматически через `EnsureCreated()`.
