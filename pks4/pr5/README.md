# Практика 5

Простой проект на `ASP.NET Core MVC` + `EF Core` + `PostgreSQL`.

Что есть:

- материалы с пополнением склада;
- продукты и привязка материалов;
- заказы с расчетом времени;
- производственные линии;
- API по методичке в простом виде.

Что сделать перед запуском:

1. В `pgAdmin` создать базу данных `pks_pr5`.
2. Если нужно, поменять логин и пароль в [appsettings.json](./appsettings.json).
3. В папке проекта выполнить `dotnet restore`.
4. Потом выполнить `dotnet run`.

Таблицы и стартовые данные создаются автоматически через `EnsureCreated()`.
